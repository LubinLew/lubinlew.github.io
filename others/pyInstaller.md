# PyInstaller

### 安装

```bash
pip install pyinstaller
```

### 将python生成为执行程序

http://c.biancheng.net/view/2690.html

支持 windows 和 linux

### 将python执行程序分解

解包后 得出的 pyc 文件。

```bash
git clone https://github.com/extremecoders-re/pyinstxtractor
cd  pyinstxtractor
python3 pyinstxtractor.py 执行程序
```
