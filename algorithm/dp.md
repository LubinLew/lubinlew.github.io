# 动态规划

## 题目特点

- 计数
  
  - 有多少种方式
  
  - 有多少种方法选出k个数使得和是 sum

- 求最值
  
  - 从左上角走到右下角路径的最大数字和
  
  - 最长上升子序列长度

- 求存在性
  
  - 取石子游戏,先手是否必胜
  
  - 能不能选出k个数使用和是 sum

-----

## 思路

- 求最值型动态规划
  - 1. 确定状态
       
       1. 最后一步
       
       2. 化成子问题
    
    2. 转移方程
    
    3. 初始条件和边界情况
    
    4. 计算顺序
