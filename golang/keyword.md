# 标识符

Go 中标识符包含下面几种，关键字、内置函数、数据类型、常量标识符、空白标识符。

## 空白标识符

Go 内置 1 个空白标识符。

| 类型  | 说明    | 备注  |
| --- | ----- | --- |
| _   | 空白标识符 |     |

空白标识符用来声明一个匿名变量，改变量在赋值表达式的左侧，通常用作占位，比如忽略函数的某个返回值。
